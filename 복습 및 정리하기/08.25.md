### this
```js
function a(){
    var bb = null;
    function aa(){
        console.log(this);
    } // this: window
    aa();
    console.log(this)
    // this: window
}
a();
```
+ console.log로 this를 확인하면 window가 나오고 그안에 a,aa 함수가 포함되어있다.
